# SilentHub

SilentHubは、静かな環境での利用を想定したオフライン専用Androidアプリです。

## 機能

### サイレント通知
- 音・バイブなし
- 画面点灯＋フルスクリーン表示
- ロック画面対応

### 連絡先統合
- 複数の連絡手段を1人にまとめる
- 対応:
  - LINE
  - X（Twitter）
  - Instagram
  - メール
  - 電話番号

### JSON保存
- 完全オフライン
- インポート / エクスポート対応
- デバイス間コピー可能

## 設計思想

- 最小限UI
- ノイズ排除
- 完全ローカル
- 拡張しやすい構造

## 技術構成

- Kotlin
- Jetpack Compose
- Material 3
- MVVM + Clean Architecture
