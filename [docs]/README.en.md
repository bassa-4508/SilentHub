# SilentHub

SilentHub is a minimal, offline-first Android utility app designed for silent environments.

## Features

### Silent Notification
- No sound, no vibration
- Screen wake-up + fullscreen display
- Works on lock screen

### Contact Hub
- Combine multiple contact methods into one
- Supports:
  - LINE
  - X (Twitter)
  - Instagram
  - Email
  - Phone

### JSON Storage
- Fully offline
- Import / Export support
- Easy cross-device transfer

## Philosophy

- Minimal UI
- No distractions
- Fully offline
- Simple and extensible

## Tech Stack

- Kotlin
- Jetpack Compose
- Material 3
- MVVM + Clean Architecture
